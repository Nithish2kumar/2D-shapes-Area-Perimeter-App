# 2D Shapes Area and Perimeter Calculator

This is a Python application built using KivyMD that calculates the area and perimeter of various 2D shapes. The application provides a user-friendly interface for users to select different shapes and input their dimensions. It then computes and displays the area and perimeter of the selected shape, making it a handy tool for students, architects, and anyone interested in geometry.

## Features
- Calculate area and perimeter for shapes like square, rectangle, circle, triangle, and more.
- Intuitive and interactive UI built with KivyMD.
- Responsive design for both desktop and mobile platforms.

## Installation
To run this application, ensure you have Python and KivyMD installed on your system:

```bash
pip install kivymd
```

## Usage
1. Launch the application.
2. Select a shape from the menu.
3. Input the required dimensions.
4. Click on the calculate button to see the results.
